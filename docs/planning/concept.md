# Project Concept: "Museum Theme"

復元ソース: project_concept_and_vision.md, design-system.md

## Mission

1677 万色の海から、あなたの「運命の色」を見つける。

## Design Philosophy

"Universal Beauty & Intellectual Curiosity"

- 静寂な美術館 (Digital Color Museum) のような没入感
- 余計なアニメーションを排除 (Static & Silent)
- 黄金比 (Golden Ratio) とフィボナッチ数列 (Fibonacci) を用いたレイアウト

## Algorithm

- 色空間: **OKLCH** (知覚的均等性を重視)
- 探索手法: **Weighted Adaptive Binary Search** (重み付き適応型二分探索)
